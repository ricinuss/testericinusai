## 🎯 Sobre

**RicinusAI** é uma interface de chat intuitiva desenvolvida pela **JatrophaLabs**, projetada para oferecer uma experiência conversacional fluida com IA. Construída com HTML5, CSS3 e JavaScript puro — leve, rápida e altamente customizável.

### ⚡ Destaques

| | |
|---|---|
| 🎨 **7 Temas** | Dark, Midnight, Forest, Sunset, Ocean, Lavender, Light |
| 🖼️ **Visão** | Anexe até 5 imagens por mensagem |
| 🧠 **Chain of Thought** | Visualize o raciocínio da IA |
| 📱 **Responsivo** | Desktop, tablet e mobile |
| 🔒 **Privacidade** | Dados locais no navegador |
| ⚡ **Streaming** | Respostas em tempo real |
